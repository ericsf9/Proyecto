Hosting de la página web generada

Estas son las instrucciones para el establecimiento del hosting

Para el establecimiento del hosting gratuito debe registrarse en github.com usando como nombre de usuario el nombre de dominio que desee para su pagina web.

A continuación debera de generar un classic token y seleccionar las casillas "repo" y "workflow" de este.

Todo ello para posteriormente introducir dicho token a continuación en su correspondiente campo.

En el ultimo campo debera de introducir el nombre de dominio que desea que debera coincidir con el nombre de usuario con el que se ha registrado.
